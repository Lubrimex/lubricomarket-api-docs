# Интеграция по API c сервисом «Лубрикомаркет»
 
Данное решение позволяет обмениваться данными между интернет-магазином и системой «Лубрикомаркет» посредством внутреннего протокола через API. 
«Лубрикомаркет» предоставляет протокол обмена, уже настроенный на получение и передачу информации из своей системы. 
Интернет-магазину необходимо также реализовать взаимодействие своего ПО с данным протоколом.
