# Список адресов доставки

Метод предназначен для получения информации о всех доступных адресах доставки в рамках договора с вашей компанией. 

## Запрос на получение списка адресов доставки
Для использования необходимо отправить <b>GET-запрос</b> на URL:

| Среда | URL |
|---|---|
| тестовая | https://dev.lubricomarket.ru/api/v1/addresses | 
| боевая | https://lubricomarket.ru/api/v1/addresses |


## Параметры запроса
Поддерживаются следующие параметры:

| № | Название поля | Описание | Тип поля | Обяз. заполн. |
|---|---|---|---|---|
| 1 | limit | Лимит, количество записей за запрос, по-умолчанию 20 | int | нет |
| 2 | offset | Смещение от начальной позиции, например: ?limit=20&offset=40 позволит получить записи с 41 по 60 | int | нет |
| 3 | search | Поиск по полю full_address | string(255) | нет |


## Ответ на запрос получения списка адресов доставки
Ответ возвращается в формате JSON:

| № | Название поля | Описание | Тип поля |
|---|---|---|---|
| 1 | results | Массив результатов | array |
| 1.1 | id | Идентификатор адреса доставки, его необходимо будет передать в параметрах запроса на создание заказа, чтобы указать адрес доставки заказа | int |
| 1.2 | full_address | Полный адрес доставки одной строкой | string(255) |
| 2 | total | Количество найденных по запросу адресов | int |
| 3 | success | Возвращается со значением false только в случае ошибки и с полем message описывающим ошибку | boolean |
| 4 | message | Описание ошибки | string(255) |

## Примеры запросов

```
https://dev.lubricomarket.ru/api/v1/addresses
```


```json
{
    "results": [
        {
            "id": "609",
            "full_address": "344065, Ростовская область, г.о. город Ростов-на-Дону, г Ростов-на-Дону, ул Троллейбусная, д. 4"
        },
        {
            "id": "668",
            "full_address": "Ростов-на-Дону, ул. Садовая 1"
        }
    ],
    "total": 2
}
```

```
https://dev.lubricomarket.ru/api/v1/addresses?search=Садовая
```

```json
{
    "results": [
        {
            "id": "668",
            "full_address": "Ростов-на-Дону, ул. Садовая 1"
        }
    ],
    "total": 1
}
```
