# Базовые типы данных

В протоколе обмена используются следующие типы данных:

<b>string(n)</b> — текстовая строка, где n — максимальное количество символов.

<b>int</b> — целочисленное значение

<b>float</b> — число с плавающей точкой, в качестве разделителя используется точка.

<b>boolean</b> — логическое значение (возможные значения: true или false).

<b>dateTime</b> - дата и время в формате ISO 8601: YYYY-MM-DDThh:mm:ss.

<b>date</b> - дата в формате ISO 8601: YYYY-MM-DD.
