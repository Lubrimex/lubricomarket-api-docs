# Информация о заказе

Метод предназначен для получения информации о заказе по уникальному идентификатору.


## Запрос на получение информации о заказе
Для использования необходимо отправить <b>GET-запрос</b> на URL:

| Среда | URL |
|---|---|
| тестовая | https://dev.lubricomarket.ru/api/v1/orders/{id} | 
| боевая | https://lubricomarket.ru/api/v1/orders/{id} |

<b>id</b> - идентификатор заказа, по которому необходима информация


## Ответ на запрос информации о заказе
Ответ возвращается в формате JSON:

| № | Название поля | Описание | Тип поля |
|---|---|---|---|
| 1 | success | Возвращается со значением true если заказ найден по запрашиваемому идентификатору и false в случае ошибки и полем message c описанием ошибки | boolean |
| 2 | message | Описание ошибки, если success = false. Пустое значение если success = true | string(255) |
| 3 | object | Массив с данными о заказе | array |
| 3.1 | id | Идентификатор заказа | int |
| 3.2 | num | Номер заказа | string(11) |
| 3.3 | organization_name | Организация-продавец | string(100) |
| 3.4 | createdon | Дата создания заказа | dateTime |
| 3.5 | updatedon | Дата обновления заказа | dateTime |
| 3.6 | cost | Общая стоимость заказа, руб. | float |
| 3.7 | status | Статус заказа | string(50) |
| 3.8 | client_inn | ИНН компании-покупателя | string(12) |
| 3.9 | client_company | Название компании-покупателя | string(255) |
| 3.10 | customer_comment | Комментарий покупателя к заказу | string(255) |
| 3.11 | agreement_name | Название договора-соглашения с покупателем | string(255) |
| 3.12 | agreement_manager | ФИО персонального менеджера | string(255) |
| 3.13 | agreement_office | Офис договора-соглашения | string(50) |
| 3.14 | agreement_storehouse | Основной склад договора-соглашения | string(100) |
| 3.15 | agreement_storehouse_uid | Идентификатор основого склада договора-соглашения | string(36) |
| 3.16 | agreement_storehouse_from | Склад с которого будет осуществляться отгрузка заказа | string(100) |
| 3.17 | agreement_storehouse_from_uid | Идентификатор склада с которого будет осуществляться отгрузка заказа | string(36) |
| 3.18 | delivery_full_address | Полный адрес доставки одной строкой | string(255) |
| 3.19 | coordination | Согласование заказа, по-умолчанию false. Если у заказа значение true, значит у покупателя имеется просроченная дебиторская задолженность и отгрузка заказа не гарантируется | boolean |
| 3.20 | products | Массив продукции заказа | array |
| 3.20.1 | title | Название продукта | string(255) |
| 3.20.2 | itemKey | Идентификатор товара | string(36) |
| 3.20.3 | article | Артикул товара | string(50) |
| 3.20.4 | price | Стоимость единицы продукции, руб. | float |
| 3.20.5 | count | Количество товара в заказе | int |
| 3.20.6 | count_unit | Единица измерения (шт.) | string(20) |
| 3.20.7 | cost | Общая стоимость данного товара в заказе | array |
