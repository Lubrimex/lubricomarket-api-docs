# Информация о адресе доставки

Метод предназначен для получения информации о адресе доставки по уникальному идентификатору. 

## Запрос на получение списка адресов доставки
Для использования необходимо отправить <get>GET</get>-запрос на URL:

| Среда | URL |
|---|---|
| тестовая | https://dev.lubricomarket.ru/api/v1/addresses/{id} | 
| боевая | https://lubricomarket.ru/api/v1/addresses/{id} |

<b>id</b> - идентификатор адреса доставки, по которому необходима информация


## Ответ на запрос получения информации о адресе доставки
Ответ возвращается в формате JSON:

| № | Название поля | Описание | Тип поля |
|---|---|---|---|
| 1 | success | Возвращается со значением true если адрес доставки найден по запрашиваемому идентификатору и false в случае ошибки и полем message c описанием ошибки | boolean |
| 2 | message | Описание ошибки, если success = false. Пустое значение если success = true | string(255) |
| 3 | object | Массив с данными о адресе доставке | array |
| 3.1 | id | Идентификатор адреса доставки | int |
| 3.2 | full_address | Полный адрес доставки одной строкой | string(255) |

## Пример запроса

```
https://dev.lubricomarket.ru/api/v1/addresses/609
```


```json
{
    "success": true,
    "message": "",
    "object": {
        "id": "609",
        "full_address": "344065, Ростовская область, г.о. город Ростов-на-Дону, г Ростов-на-Дону, ул Троллейбусная, д. 4"
    }
}
```
