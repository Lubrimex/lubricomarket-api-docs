# Описание сервиса
Сервис информационного взаимодействия реализует набор методов, необходимых для передачи и синхронизации данных между сервисом «Лубрикомаркет» 
и клиентом интернет-магазином или партнером, представляющим интернет-магазин.

Взаимодействие с сервисом происходит в режиме онлайн с помощью стандартного протокола HTTPS и формата JSON.

Инициатором запросов является как ИС клиента, так и «Лубрикомаркет» (в формате вебхуков).

## Правила формирования сообщений
При формировании сообщения необходимо руководствоваться следующими правилами:

- В заголовке обязательно необходимо передавать токен авторизации.
- Обмен сообщениями должен осуществляться в кодировке UTF-8.
- Если элемент сообщения является необязательным, то такой элемент в сообщении может отсутствовать, либо значение элемента может быть пустым.
- Если элемент сообщения является обязательным, то он не может быть пустым.
